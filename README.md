# VideoBot
刷zjooc水课视频

可以使用docker部署

* 没有设置倍速,可以添加这个功能(但是我懒)



# 本地部署

1. 使用以下命令设置环境

```shell
pip install -r requirements.txt
```

2. 安装**chrome**和**chromedriver**.(学习博客配置)

3. 使用`python main.py`运行

* 输入账号密码即可
